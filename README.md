
# Robot Antasena dan Gatotkaca [2026]

Antasena dan Gatotkaca adalah nama dua robot yang akan digunakan untuk mewakili Universitas Sebelas Maret di Kompetisi Robot Abu Indonesia. Robot di dalam repositori ini adalah kode sumber untuk robot tahun 2026. Kode dibagi menjadi beberapa bagian besar yaitu *movement*, *tools*, dan *image processing*.

## 🚀 Fitur

* **Movement** 
- FreeRTOS
- x-drive dan mecanum purpose
- Implementasi PID
- Implementasi OOP
- Sistem kendali loop tertutup
    - Encoder 2 fase
    - IMU MPU6050
- Menggunakan 8 dan bit PWM
- Log untuk Serial Studio
